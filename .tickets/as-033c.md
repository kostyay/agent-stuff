## Context
`dashboard.ts` renders agent status cards for the live progress widget. It exports `estimateContextWindow` (pure lookup) and `renderCard` (renders a 6-line box-drawing card). Imports `ThemeColor` type, `expandTabs`/`formatTokens`/`truncate` from `./formatting.js`, `agentColor` from `./tui-helpers.js`, and `RunState` type from `./types.js`.

## Functions to test

### `estimateContextWindow(model)`
- `"claude-3-opus"` → `200_000`
- `"gpt-4o-mini"` → `128_000`
- `"gpt-4-turbo-preview"` → `128_000`
- `"o1-preview"` → `200_000`
- `"o3-mini"` → `200_000`
- `"o4-mini"` → `200_000`
- `"gemini-1.5-pro"` → `1_000_000`
- `"deepseek-chat"` → `128_000`
- `"unknown-model-xyz"` → `null`
- `undefined` → `null`
- Case insensitive — `"Claude"` → `200_000`

### `renderCard(state, colWidth, theme, displayIndex)`
- Returns array of exactly 6 strings (top border, name line, description, status, context bar, bottom border)
- **Status icons**: `"running"` → `●`, `"done"` → `✓`, `"aborted"` → `⊘`, `"error"` → `✗`
- **Context bar**: when model has known context window and `contextTokens > 0`, shows `[##---] XX%` format
- **Context bar fallback**: unknown model with tokens → `ctx: Xk` format
- **No context**: `contextTokens = 0` → `—`
- **Step label**: `state.step = 3` → includes `#3` in status line
- **Description override**: `state.description` set → shows description text instead of `lastLine`
- **Card width**: respects `colWidth` parameter (content = colWidth - 2)
- **Box drawing**: top line starts with `┌`, bottom with `└`
